# Gomoku-Complete
A complete front end and back end of the game Gomoku

## To start the server

In the project terminal, you can run:

### `npm run dev`

PORT: http://localhost:8080/

## To start the app

 In the project terminal, you can run:

### `npm start`

Visit http://localhost:3000/
